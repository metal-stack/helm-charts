# CSI Driver LVM Helm Chart

This is the helm chart for the deployment of https://github.com/metal-stack/csi-driver-lvm.

The source of this helm-chart is [csi-driver-lvm](https://github.com/metal-stack/csi-driver-lvm) and gets synced to [helm-charts](https://github.com/metal-stack/helm-charts) after a release.

**The chart requires that you have at least Kubernetes 1.16.**
